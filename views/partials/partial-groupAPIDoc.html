Sentry Groups API Docmunetation

GET:       /groups       Gets information of all groups.

Parameters Type Description Location Examples
None N/A No Parameters Needed. N/A N/A

POST:       /groups       Creates a new group.

Parameters Type Description Location Examples
groupid string The unique ID of the group. Created by the controller body 7F72822C7F3E, 92ACABEF843E, 8B511BCDC0C9
groupname string The name of the group. body Emerald Dwarves, Sentry Team, Louieville Sluggers
maxsize integer The max amount of members allowed in the group. body 2, 5, 25, 18
